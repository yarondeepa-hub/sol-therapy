# Sol Therapy Website - Spec v4

> **מבנה:** מאיפיון האתר המקורי (`B-brain/data/אפיון-אתר-sol-therapy.md`)
> **שפה ויזואלית:** מלוח ההשראות Pinterest (37 פינים) + 6 אתרי רפרנס תרבותיים
> **פלטה:** מהאיפיון המקורי (Sol Green + Beni + Washi)
> **אנימציות + טיפוגרפיה + layouts:** מאתרי הרפרנס (migdalor, exhibition, telavivdance, rotemcohensoaye)

---

## 1. מהות הפרויקט

**חזון:** האתר הוא לא ויטרינה - הוא חלק מהחוויה. כניסה לאתר צריכה להרגיש כמו כניסה למרחב מדיטטיבי. נייר Washi, דיו שזורם, שקט שמדבר.

**שלוש מטרות:**
1. **חוויה אמנותית** - האתר עצמו הוא יצירה
2. **קידום אירועים** - מכירת כרטיסים
3. **סמכות מותגית** - ביסוס מעמד כמוביל בתחום

### Section 1: Hero - כניסה למרחב

**רקע:** אינדיגו כהה (#264348) - הסקשן היחיד שהוא כהה-first

**תוכן:**
- לוגו Sol Therapy בפינה העליונה
- כותרת "סול תרפי" - Frank Ruhl Libre, ענקית (100-120px clamp), צבע Washi cream
- תגלית "היכן שהצליל הופך לדממה" - מתגלה בעיכוב
- אינדיקטור גלילה - טיפת דיו שנופלת לאט

**אנימציה:** Ink Reveal - הכותרת מתגלה כאילו מישהו כותב אותה בדיו. קליגרפיה שמתפתחת לאט.

**מעבר לסקשן הבא:** גרדיינט בוקאשי מאינדיגו לקרם Washi - כמו שחר שעולה

**מה זה לא:** Stock photo hero, gradient overlay גנרי, "ברוכים הבאים"

**מה זה כן:** הביטחון של הטייפ הענק של migdalor + אנימציית דיו ייחודית + רגע דרמטי שמושך פנימה

### Section 2: התדר שלנו (אודות)

**רקע:** Washi cream (#F2EAD3)
**צבע טקסט:** Sol Green (#588475)

**עיצוב:** טיפוגרפיה גדולה, אוצרותית, כמו טקסט תערוכה במוזיאון. יישור לא סימטרי - הטקסט לא במרכז, הוא נשען לצד כמו טקסט על קיר גלריה.

**מבנה:**
- שורה 1: מה זה (חוויות שמע אימרסיביות במרחבים תרבותיים)
- שורה 2: מה זה לא (לא טיפול קליני. לא מדיטציה מודרכת.)
- 2-3 שורות: מה הקהל חווה (הקשבה, גוף ככלי תהודה, קשב משתנה)
- "קרא עוד" שפותח מודל עם הפילוסופיה המלאה

**אנימציה:** Sumi-e Scroll - אלמנטים מתגלים בגלילה כמו משיכות מכחול, מקל למעובה. טקסט fade-up word by word (50ms stagger).

**Layout:** אסימטרי 60/40. טקסט בצד אחד, איור Sumi-e עדין (בונסאי או במבוק מלוח הפינטרסט) נשפך מהקצה השני.

### Section 3: מסעות קרובים (אירועים)

**רקע:** Washi cream (#F2EAD3)
**צבע טקסט:** Sol Green (#588475)
**צבע אקסנט:** Beni (#BE5069) לתאריכים ול-CTA

**עיצוב:** רשימה טיפוגרפית נקייה - כמו קטלוג תערוכה. לא כרטיסים עם תמונות - טקסט בלבד עם קווים מפרידים עדינים.

**כרטיס אירוע (Catalog Entry):**
```
תאריך (Beni)  |  שם האירוע (Sol Green, serif heavy)  |  מיקום
               |  פורמט - משך - קיבולת (muted)        |
               |  [שמרו לי מקום] (CTA)                |
```

**כפתור רכישה:** "שמרו לי מקום" - צבע Beni (#BE5069)
**מצב sold out:** מעבר אוטומטי לרשימת המתנה

**פס אמון מקוצר (תחת האירועים):**
שורה עדינה מונוכרומטית של לוגואים: מוזיאון ת"א, סוזן דלל, הספריה הלאומית, מלון פסטורל.

**אנימציה:** כל entry מחליק פנימה מהצד (מתחלף ימין/שמאל). קווים מפרידים גדלים ברוחב בגלילה.

**טכני:** דף קנוני לכל אירוע: `/אירועים/<slug>`. כל לחיצה על שם אירוע מובילה לדף מלא.

### Section 4: ארכיון ויזואלי (גלריה)

**רקע:** Washi cream (#F2EAD3)

**עיצוב:** גריד אורגני - לא סימטרי, לא מסודר. תמונות בתוך צורות "Zen Pebble" - blobs אורגניים. שילוב של סטילס ווידאו-שורטס בלופ.

**מטאדאטה אוצרותית:** כיתוב ב-hover/tap: `מקום / עיר / שנה`
**פילטור:** מוזיאונים | ריטריטים | ליווי מוזיקלי | קהל

**אנימציה:** Zen Pebble Float - צורות אורגניות שצפות עם צל רך. תמונות מתגלות scale 0.95 to 1.0 + opacity. Lightbox נגיש עם focus trap ו-swipe במובייל.

## 4. דפים נוספים

### דף אירוע (`/אירועים/<slug>`)

דף שלם לכל אירוע - קריטי ל-SEO ולשיתוף.

**מבנה:**
- Hero עם תמונת האירוע (Zen Pebble mask)
- שם האירוע + תאריך + מיקום
- תיאור מורחב
- לינאפ אמנים עם כרטיסי מציג מוקטנים
- CTA ראשי - "שמרו לי מקום"
- מפת מיקום
- Schema.org: Event + Offer

### יומן (`/יומן`)

ארכיון תוכן אוצרותי - לא "בלוג". כל פיסה קצרה, מובלת-תמונה, ציטוט אחד, רעיון אחד.

**סוגי תוכן:** מאמרים על ריפוי בצליל, סיקור אירועים, ראיונות עם אמנים, רגעים מהשטח.

### ערכת עיתונות (`/לעיתונות`)

דף ייעודי לעיתונאים, אוצרים ומוסדות. טקסט תקני + תמונות + לוגואים + ציטוט חזק.

## 7. אנימציות - דיו במים

**עיקרון:** כל תנועה באתר צריכה להרגיש כמו דיו שזורם בנייר רטוב - אורגנית, איטית, בלתי הפיכה. לא bouncing. לא sliding. התגלות.

| אנימציה | איפה | תיאור |
|----------|------|--------|
| Ink Reveal | Hero | הכותרת מתגלה כאילו מישהו כותב אותה בדיו |
| Sumi-e Scroll | כל סקשן | אלמנטים מתגלים בגלילה כמו משיכות מכחול |
| Bokashi Fade | מעברי סקשנים | גרדיינטים חלקים כמו טכניקת בוקאשי |
| Zen Pebble Float | גלריה | צורות אורגניות שצפות עם צל רך |
| Ink Drop | אינדיקטור גלילה | טיפת דיו שנופלת לאט |
| Washi Texture | רקע | טקסטורת נייר עדינה שנעה מעט בגלילה (parallax) |
| Word Stagger | כותרות | מילים מופיעות אחת אחרי השנייה (50ms stagger) |
| Seal Press | חותמות Hanko | Scale מ-0.8 ל-1.0 + סיבוב קל שמתיישב |
| Line Grow | מפרידים | קווים גדלים ברוחב/גובה בגלילה |
| CTA Fill | כפתורים | Beni fill סוחף מצד אחד ב-hover |

**כללים:**
- כל אנימציה מכבדת `prefers-reduced-motion`
- לא יותר מאנימציה אחת פעילה באותו viewport
- מהירות: 0.8s-1.5s, `cubic-bezier(0.23, 1, 0.32, 1)`
- אנימציה פעם אחת בלבד (לא חוזרת בכל scroll)

## 8. טקסטורה ואלמנטים ויזואליים

צורת "אבן נחל" אורגנית - blob mask לתמונות:
- לא עיגול, לא ריבוע - צורה טבעית מעוגלת
- כל שימוש עם וריאציה שונה קלות
- צל רך מתחת (floating effect)
- שימוש: תמונות גלריה, avatar אמנים

## 11. דרישות טכניות

### Stack

| שכבה | טכנולוגיה |
|------|-----------|
| Framework | Astro 5.x (Zero JS by default, View Transitions) |
| CMS | Sanity (Headless, ויזואלי, מצוין לעברית) |
| Hosting | Netlify / Vercel |
| אנימציות | GSAP + ScrollTrigger |
| Styling | CSS custom properties (Design Tokens) |
| Images | Astro Image + Sanity CDN (WebP/AVIF, lazy-load) |
| Analytics | Plausible (פרטיות-first) |

**הערה חשובה:** ה-build הראשון יהיה HTML/CSS/JS סטטי (כמו שעבדנו עד עכשיו) כדי לאשר את העיצוב. רק אחרי אישור ויזואלי - נעבור ל-Astro + Sanity.

### נגישות (WCAG AA)
- ניגודיות מינימלית 4.5:1 לטקסט רגיל, 3:1 לכותרות
- Sol Green Dark (#3B514B = 6.2:1) לטקסט קטן על Washi
- ניווט מקלדת מלא, visible focus states
- `prefers-reduced-motion` - כל האנימציות נכבות
- Skip-to-content link
- `lang="he"` + `dir="rtl"`

### Responsive

| Breakpoint | רוחב | שינויים |
|------------|-------|---------|
| Desktop | > 991px | layouts אסימטריים מלאים, טייפ ענק |
| Tablet | <= 991px | רשתות פשוטות, טייפ קטן יותר |
| Mobile landscape | <= 767px | עמודה אחת |
| Mobile portrait | <= 478px | מובייל מלא, touch targets גדולים |
