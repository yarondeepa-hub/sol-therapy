### הטכניקה החתימתית: Risograph-to-Digital Pipeline

זה הליבה של מה שמבדיל את טל: הוא סורק טקסטורות ריזוגרף פיזיות ומשלב אותן בקומפוזיטינג דיגיטלי. התוצאה - גרעיניות, תוססות צבע, והיסטים של שכבות צבע שמייצרים את המראה האנלוגי החם הזה, אבל בתנועה.

**הפייפליין:**
1. Illustrator/Photoshop - יצירת אלמנטים גרפיים שטוחים, סריקת טקסטורות ריזו
2. Cinema 4D - בניית אובייקטים תלת-ממדיים, מיפוי טקסטורות שטוחות על משטחים
3. After Effects - קומפוזיטינג שכבות, אנימציה, הוספת גרעיניות
4. יצוא כוידאו (WebM לרשת, MP4 לסושיאל, GIF לשיתוף מהיר)

### אופני תנועה

**A. סיבוב/מסלול (Turntable)**
אובייקט מרכזי (גביע, מזרקה, מבנה דמוי עוגה) מסתובב לאט על ציר אנכי. הקמרה קבועה. האובייקט מתגלה דרך הסיבוב. התנועה קונטמפלטיבית, כמעט טקסית.

**B. אנימציית שכבות שטוחות (Risograph Shuffle)**
שכבות גרפיות שטוחות זזות באופן עצמאי: רקע גולל שמאלה, צורות ביניים פועמות, טקסט בחזית מקפץ. כל שכבה בקצב משלה - בדיוק כמו שדפוסי ריזוגרף נבנים שכבה-שכבה.

**C. מעברי סצנות נרטיביים**
אנימציה מבוססת סצנות עם תנועות מצלמה, מעברי dissolve, ושינויי פרספקטיבה. רקעים תלת-ממדיים עם טקסטורות ריזוגרף. דמויות דו-ממדיות שטוחות זזות במרחב תלת-ממדי.

**D. טיפוגרפיה קינטית**
אותיות מונפשות כאובייקטים עצמאיים - הן מתרחבות, מסתובבות, מתמורפפות ומתערמות. הטיפוגרפיה הופכת לכוריאוגרפיה.

**E. אנימציית דפוס/טקסטורה**
עדינה אבל קריטית: דפוסי רקע זזים, טקסטורות מתנודדות, גרעינים רוטטים. נותן לאלמנטים "סטטיים" איכות חיה.

### קצב ומקצב

**מדוד, מכוון, היפנוטי.** לא מהיר. לא קופצני.

- **easing חלק** - אובייקטים לא קופצים. הם מחליקים. cubic-bezier שמרמז על משקל וכבידה
- **קצב מבוסס לופים** - רוב האנימציות 4-15 שניות. נקודת הלופ חלקה. יוצר איכות מדיטטיבית ואינסופית
- **ללא חתכים חדים** - מעברים נמסים, cross-dissolve, מורפינג
- **"נשימה" ברקעים** - דפוסים שפועמים לאט, יוצרים תחושה שכל הקומפוזיציה חיה

### היחס בין תנועה לרגש

- **סיבוב איטי = יראת כבוד.** אובייקטי Hermes מסתובבים כמו יצירות מוזיאון על פודסטל
- **סחף שכבות = חלימה.** שכבות עצמאיות בקצבים שונים יוצרות עומק פרלקס שמרגיש כמו סחף בחצי-שינה
- **פעימת טקסטורה = חיים.** הגרעיניות הרוטטת אומרת: זה לא דפוס מת, זה חי, יש לו דופק
- **הרכבה טיפוגרפית = הכרזה.** כשטקסט בונה את עצמו, נוצרים ציפייה והגעה

### מה ניתן לאמץ (עם התאמה)

1. **טכניקת "הנשימה הסמויה"** - גרעיניות רוטטת על קומפוזיציות "סטטיות". הדבר הכי טרנספרי. כל איור סומי-אה באתר יכול לקבל overlay של 4 שניות של grain loop. האפקט: הנייר שעליו הדיו נמצא נראה חי.

2. **סיבוב turntable איטי** - אנסו או קערת סאונד, מרופד בטקסטורת וואשי, מסתובב בקצב כמעט בלתי נראה. הצופה מגלה את התנועה רק אחרי כמה שניות.

3. **פרלקס שכבות עצמאי** - ההרים המרובדים שלנו (חזית כהה, ערפל ביניים, פסגות רקע) עם scroll-triggered parallax עדין.

4. **הרכבה טיפוגרפית** - טקסט שמרכיב את עצמו תו-תו, כמו משיכות מכחול שנכתבות בזמן אמת.

5. **טקסטורת ריזוגרף על 3D** - מוטיבים יפניים מסורתיים (דפוסי גלים, אנסו, מחטי אורן) כאובייקטי wireframe תלת-ממדיים עדינים ב-Three.js.

## 6. Implementation Roadmap - מפת דרכים ליישום

### Phase 1: מה שאנחנו יכולים היום (GSAP - כבר מותקן)

כל אנימציית סקרול באתר aroke01 ניתנת ליישום עם GSAP שלנו:

**Fade-up entrance:**
```javascript
gsap.from('.gallery-item', {
  y: 40, opacity: 0, duration: 0.8,
  ease: 'power2.out', stagger: 0.1,
  scrollTrigger: { trigger: '.gallery-grid', start: 'top 80%' }
});
```

**Blur-in reveal:**
```javascript
gsap.from('.hero-image', {
  filter: 'blur(8px)', duration: 1.2,
  ease: 'power2.out',
  scrollTrigger: { trigger: '.hero-image', start: 'top 75%' }
});
```

**3D Helix scroll:**
```javascript
gsap.fromTo('.card', {
  rotateY: 60, rotateZ: 2, transformPerspective: 3000
}, {
  rotateY: -60, rotateZ: -2, transformPerspective: 3000,
  ease: 'none',
  scrollTrigger: { trigger: '.card', scrub: true }
});
```

**Loose Hinge:**
```javascript
gsap.fromTo('.hanging-element', {
  rotation: 10, transformOrigin: '0% 0%'
}, {
  rotation: -10, transformOrigin: '100% 100%',
  ease: 'sine.inOut',
  scrollTrigger: { scrub: true }
});
```

**Text assembly (SplitText):**
```javascript
const split = new SplitText('.section-title', { type: 'chars' });
gsap.from(split.chars, {
  opacity: 0, y: 20, duration: 0.5, stagger: 0.03,
  ease: 'power2.out',
  scrollTrigger: { trigger: '.section-title', start: 'top 80%' }
});
```

## 12. Performance & Best Practices

### לקחים מ-aroke01.com

**מה לחקות:**
- CSS-only animations כשאפשר (אפס עלות runtime)
- will-change: filter על אלמנטים מונפשים (רמז ל-GPU)
- רק transform ו-filter ב-keyframes (מאפייני compositing - לא layout thrashing)
- autoplay videos: muted + looped (אין overhead של audio decode)

**מה לא לחקות:**
- ~990MB משקל אתר (לא מתקבל על הדעת)
- אין prefers-reduced-motion (אנחנו חייבים)
- אין picture/AVIF/WebP (אנחנו כבר עושים את זה)
- 14 autoplay videos בדף הבית (bandwidth וסוללה)

### Best Practices שלנו (כבר קיימות)

- prefers-reduced-motion support
- picture עם AVIF + WebP fallback
- GSAP animations על requestAnimationFrame (60fps composited)
- ScrollTrigger lazy by default
- loading="lazy" על תמונות
