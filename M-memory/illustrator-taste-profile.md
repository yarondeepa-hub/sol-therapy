## Taste Spectrum - ציר הטעם

| ציר | ירון נוטה ל... | לא ל... |
|-----|----------------|---------|
| מופשט vs פיגורטיבי | **פיגורטיבי-אמנותי** - נופים, הרים, גבעות | הפשטה טהורה (כתמי דיו ללא צורה) |
| מינימלי vs עשיר | **עשיר בשכבות** - עומק, שכבות חופפות, ווריאציית צבע | שטוח, חד-ממדי, מעט אלמנטים |
| כהה vs בהיר | **שניהם** - אינדיגו כהה ל-hero, ירוק-קרם ל-about | אחיד - לא הכל כהה ולא הכל בהיר |
| מסורתי vs מודרני | **מסורתי עם מגע מודרני** - טכניקות Sumi-e/Shin-hanga | גנרי-דיגיטלי, stock art feel |
| שקט vs דרמטי | **דרמטי** - ערפל, מיסט, עומק אטמוספרי | שקט מדי, ריק מדי |
| עמוס vs נושם | **נושם** - 60-75% חלל ריק, אלמנט מרכזי אחד | עמוס, cluttered, יותר מדי פרטים |
| מבריק vs מאט | **מאט** - נייר, דיו, חומריות ידנית | מבריק, glossy, 3D, פלסטי |


## Prompt Engineering Rules - כללי prompt

### חובה לכלול:
```
- "warm paper texture" / "warm off-white paper"
- "monochrome ink wash" / "visible brushwork"
- "zen negative space" / "lots of breathing room"
- "one focal subject" / "single central element"
- "traditional [technique] painting style"
- "fine art museum quality" / "museum-catalog feel"
- "subtle paper grain" / "paper texture visible"
- "quiet mood" / "contemplative atmosphere"
- "no text no letters no numbers" (למניעת artifacts)
```

### אסור לכלול (הוסר 15.02):
```
- NO "vermilion seal" / "red seal stamp" / "hanko" - ירון לא רוצה חותמת
```

### חובה להימנע:
```
- "minimal" / "simple" / "sparse" (ירון רוצה עשיר בטקסטורה, לא ריק)
- "abstract" ללא צורות מזוהות
- "flat" / "clean" (ירון רוצה טקסטורה)
- כל מספר/טקסט (נוטה להיכשל)
- hex codes כטקסט בprompt (מתרנדר כטקסט)
- "glossy" / "3D" / "neon" / "gradient" (מנוגד לאסתטיקה)
- "photorealism" / "photographic" (הסגנון הוא ציורי)
- "clutter" / "crowded" / "detailed background"
- "decorative fonts" / "large titles"
```

### Template A - Hero (dark, dramatic):
```
dramatic [indigo/navy] mountain landscape painting, traditional East Asian
ink wash technique, towering peaks with [snow/mist] emerging from thick
atmospheric clouds, multiple receding mountain layers creating extreme depth,
rich tonal gradation from near-black foreground to misty background, visible
wet brushstroke texture, paper grain visible, fine art museum masterwork
quality, no text, 8k ultra detailed
```

### Template B - About (warm, green):
```
lush green rolling hills watercolor painting, multiple overlapping organic
hill shapes in varied shades of green from deep forest to light sage,
Japanese nihonga painting technique, each hill layer has different saturation
creating rich visual depth, soft watercolor bleeding edges, warm cream paper
background showing through, subtle paper grain texture, fine art gallery
quality, no text, 8k detail
```

### Template C - Zen Poster (seal + negative space):
```
Short: Minimal Zen poster on warm paper, single black brushstroke subject,
lots of negative space, calm editorial layout.

Long: Create a vertical poster on warm off-white paper texture. Place a
single expressive black/charcoal brushstroke subject slightly off-center.
Keep 70% negative space, no gradients, no glossy effects. Optional
micro-caption in tiny uppercase sans near bottom margin. Clean edges,
subtle paper grain, quiet mood, museum-catalog feel. No text in image.
```

### Template D - Ink Landscape (mountain/mist/water):
```
Short: Sumi-e ink-wash landscape, misty mountain and water, monochrome
washes, minimal lines, tranquil atmosphere.

Long: Paint an ink-wash landscape inspired by East Asian monochrome
traditions: distant mountain forms dissolving into mist, quiet water plane
in foreground, layered washes and soft edges. Mostly grayscale and paper
tones. No photorealism; capture essence with minimal strokes and controlled
emptiness. Vertical scroll feel, horizon high, soft gradient wash.
Contemplative, sparse, timeless. No text.
```

### Template E - Blue Topography (traditional-modern hybrid):
```
Short: Indigo topographic texture with river-delta lines, printmaking look,
muted palette, minimal caption, calm design system.

Long: Generate a blue/indigo print-like composition resembling a topographic
river delta: branching channels, contour-like textures, high detail but
muted contrast. Clean and editorial layout: generous margins, minimal
typography only if needed (tiny uppercase sans, neutral ink). Texture:
etched/woodcut feel, no neon, no heavy saturation. Vertical poster or
square cover, serene and sophisticated. No text in image.
```

### Master Prompt Template (for any subject):
```
Style: warm paper texture, monochrome ink wash, zen negative space,
minimal editorial layout.
Palette: warm off-white paper, charcoal ink (optional: indigo or teal).
Composition: 70% negative space, one focal subject, vertical scroll feel.
Avoid: glossy gradients, neon, photorealism, clutter, decorative fonts,
heavy noise.
Subject: <SUBJECT>.
```

## Learning Loop Insights (15.02.2026) - Integration Reset

### What went wrong with v1 decorations:
- Opacity 0.05-0.12 = invisible. Yaron called them "faded"
- Generic quality - prompts not detailed enough
- Ghost-background approach (absolute positioning behind text) = wrong concept entirely
- Yaron wants illustrations VISIBLE and BEAUTIFUL, not hidden
