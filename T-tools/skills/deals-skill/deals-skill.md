# Deals Skill (Deal Desk Hotels)

מספרים, סיכונים, והצעה נגדית שאפשר לחתום עליה.

## When to Use

- מלון/שותף פנה עם הצעה לשיתוף פעולה
- יש רעיון לריטריט חדש וצריך לבנות מודל כלכלי
- קיבלתם הצעה בטלפון וצריך להפוך אותה למספרים
- רוצים להשוות בין כמה הצעות/מיקומים
- צריך להכין הצעה נגדית למשא ומתן

---

## Inputs

### Required (חובה)

| Input | Description | Example |
|-------|-------------|---------|
| **מיקום** | שם המלון/מקום | פסטורל כפר בלום |
| **תאריכים** | טווח תאריכים | 23-26 אפריל 2026 |
| **תפוסה מקסימלית** | כמה אנשים/חדרים | 196 חדרים / 400 אורחים |
| **מודל כלכלי** | סוג העסקה | MOA (50/50) / Pastoral (תוכן + עמלה) |

### Optional (אופציונלי)

| Input | Description | Default |
|-------|-------------|---------|
| **מחיר כרטיס בסיס** | מחיר לאורח | ייגזר מהוצאות + מרווח |
| **הוצאות ידועות** | עלויות שכבר מוכרות | — |
| **הסכמות קיימות** | מה כבר סוכם בשיחה | — |
| **דרישות מיוחדות** | תנאים ספציפיים | — |
| **הצעה מתחרה** | הצעה ממיקום אחר להשוואה | — |

### 2. Scenario Model (Google Sheet)

**הנכס הכי חשוב** — מייצר אמת מספרית עם נוסחאות עקביות.
כל שינוי קטן (תפוסה/מחיר/הוצאה) מראה מיד את התוצאה.

#### Tab Structure (10 טאבים)

| Tab | Name | Purpose |
|-----|------|---------|
| 00 | **Inputs** | כל המספרים והנחות, במקום אחד |
| 01 | **Revenue** | הכנסות לפי מנגנון הדיל |
| 02 | **Costs** | הוצאות לפי בלוקים |
| 03 | **PnL** | רווח/הפסד + חלוקה לשותפים |
| 04 | **Scenarios** | נמוך/בסיס/גבוה + טבלת השוואה |
| 05 | **Breakeven** | נק' איזון + שיטות בדיקה |
| 06 | **Sensitivity** | מה קורה אם... (מחיר/תפוסה/הוצאות) |
| 07 | **Cashflow** | מתי כסף נכנס ומתי יוצא |
| 08 | **Risks** | סיכון/השפעה/הסתברות/מיגון |
| 09 | **Versions** | לוג גרסאות ושינויים |

## Business Models Reference

### MOA Model (ריטריט קטן)

```
Revenue: 100%
 └── Expenses: ~270,000 ₪
      ├── Hotel: ~150,000 ₪
      └── Content: ~120,000 ₪
 └── Net Profit: Revenue - Expenses
      └── Split: 50% / 50%
```

### Pastoral Model (ריטריט גדול)

```
Revenue to Sol Therapy:
 ├── Fixed Content Fee: 130,000 ₪
 └── Commission per Room:
      ├── First 100 rooms: 650 ₪ × 100 = 65,000 ₪
      └── Remaining 96 rooms: 700 ₪ × 96 = 67,200 ₪

Total (at 196 rooms): 262,200 ₪
```

### Break-Even Formula

```
Break-Even Point = Fixed Costs / (Revenue per Unit - Variable Cost per Unit)

Example:
Fixed Costs = 150,000 ₪
Revenue per Room = 3,500 ₪
Variable Cost per Room = 1,500 ₪
Contribution Margin = 2,000 ₪

Break-Even = 150,000 / 2,000 = 75 rooms
```

## Quality Checklist

### Before Presenting Deal

- [ ] Deal Folder קיים עם מבנה נכון
- [ ] כל ה-Inputs מלאים בSheet
- [ ] 3 תרחישים מחושבים (Low/Base/High)
- [ ] Break-even מחושב (תפוסה + מחיר)
- [ ] Sensitivity נבדק
- [ ] Cashflow מיופה
- [ ] סיכונים מזוהים עם מענים
- [ ] היסטוריית גרסאות מעודכנת
- [ ] המלצה ברורה (GO/NO-GO/תנאי)

### Before Negotiation

- [ ] BATNA מוגדר
- [ ] Deal Breakers ברורים (5 תנאים קריטיים)
- [ ] Trading Chips מוכנים
- [ ] הצעה נגדית כתובה (5-10 סעיפים)
- [ ] מספרים בראש
- [ ] יודע מה "הצלחה" עבורם

### After Call

- [ ] Summary Email נשלח תוך 24 שעות
- [ ] גרסה עודכנה בכל המסמכים
- [ ] נושאים פתוחים מתועדים

### Before Signing

- [ ] כל התנאים מתועדים ב-One-Pager
- [ ] חוזה נבדק משפטית
- [ ] ביטוחים בסדר
- [ ] לו"ז תשלומים ברור
- [ ] גרסה סופית מאושרת
