# Location Research Workflow

> תהליך עבודה לחיפוש מיקומים חדשים לאירועים וריטריטים של סול תרפי.

---

## סקירת התהליך

```
[אתם] מגדירים את הצורך (סוג אירוע, אזור, תקציב)
          ↓
[Researcher] מחפש ומעריך מיקומים
          ↓
[Gatekeeper] בודק התאמה למותג
          ↓
[אתם] מחליטים ויוצרים קשר
          ↓
[Copywriter] כותב תוכן על המיקום (אם נבחר)
```

---

## שלב 1: הגדרת הצורך

### לפני שמתחילים

תנו ל-Researcher את המידע הבא:

| שאלה | דוגמה |
|------|-------|
| **סוג אירוע** | ערב / ריטריט / אירוע מיוחד |
| **אזור מבוקש** | מרכז / צפון / דרום / גמיש |
| **כמות משתתפים** | 50-100 / 100-200 / אחר |
| **תקציב משוער** | נמוך / בינוני / גבוה / לא מוגדר |
| **דרישות מיוחדות** | לינה, קייטרינג, נוף, אחר |

---

## שלב 2: Researcher מחפש

### מה הסוכן עושה

1. **קורא את C-core** — מבין את האסתטיקה של המותג
2. **מחפש מקומות** — לפי הקריטריונים שהוגדרו
3. **מעריך כל מקום** — אסתטיקה, אקוסטיקה, נגישות, התאמה
4. **כותב דוח** — עם המלצות וצעדים הבאים

### פורמט פלט

```markdown
# Location Research: [נושא]

**תאריך:** [תאריך]
**סוג אירוע:** [ערב / ריטריט / מיוחד]

---

## סיכום מנהלים
[2-3 משפטים]

## מקומות שנמצאו
[רשימה מוערכת]

## המלצה
[מקום מומלץ + נימוק]

## צעדים הבאים
[מה לעשות עכשיו]
```

### שמירת קבצים

```
O-output/
└── [XX]-location-research-[נושא]/
    └── researcher-report.md
```

---

## שלב 3: Gatekeeper בודק

### מה שומר הסף בודק

| קריטריון | שאלה |
|----------|------|
| התאמה למותג | האם המקום מתאים לאסתטיקה של סול תרפי? |
| שלמות המחקר | האם כל הקריטריונים נבדקו? |
| איכות ההמלצה | האם יש המלצה ברורה עם נימוק? |
| צעדים הבאים | האם ברור מה לעשות עכשיו? |

### פלט

- **APPROVED** — המחקר מלא ומועיל
- **NEEDS MORE** — צריך לבדוק עוד אפשרויות / מידע
- **ESCALATE** — צריך החלטה אנושית

---

## שלב 4: החלטה ויצירת קשר

אתם מחליטים:
- לפנות למקום המומלץ
- לבקש מחקר נוסף
- לבקר פיזית במקום

---

## שלב 5: תוכן (אופציונלי)

אם המקום נבחר לאירוע, Copywriter יכול:
- לכתוב תיאור המקום לאתר/רשתות
- לכתוב הכרזה על האירוע במיקום החדש

## מתי להשתמש ב-Workflow הזה

✅ **כן:**
- מחפשים מקום חדש לאירוע
- רוצים להרחיב לאזור חדש
- צריכים אלטרנטיבה למקום קיים

❌ **לא:**
- המקום כבר ידוע ונבחר
- רק צריכים פרטי קשר של מקום ספציפי
- שאלה מהירה על מקום מוכר
