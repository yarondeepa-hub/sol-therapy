## הבעיה שפתרנו

כל צ'אט מתחיל מאפס.
אתם מסבירים את המותג שלכם. **שוב.**
אתם מסבירים את הקהל שלכם. **שוב.**

## מערכת ABC-TOM

"Feed the ABC. Let TOM execute."
תאכילו את ה-ABC. תנו ל-TOM לבצע.

### ABC = הבסיס (מה שאתם מזינים למערכת)

**A = Agents (סוכנים)**
- מי עושה מה (קופירייטר, מעצב, חוקר, גייטקיפר...)
- ההוראות שלהם (במה הם טובים, איך הם עובדים)
- ההיררכיה (אתם ה-CEO. הסוכנים מבצעים. אתם מחליטים.)

**B = Brain (מוח)**
- דוגמאות כתיבה שמלמדות את הקול והסגנון שלכם
- מחקר ומידע (תובנות מהתעשייה, נתונים, עובדות)
- חומרי עזר (כל מה שהסוכנים צריכים לדעת)

**C = Core (ליבה)**
- Project Brief (מה אתם עושים, את מי אתם משרתים, המשימה שלכם)
- Voice DNA (איך אתם נשמעים, טון, אוצר מילים)
- ICP Profile (הלקוח האידיאלי שלכם, הבעיות שלו, השפה שלו)

### TOM = הביצוע (מה שהמערכת עושה)

**T = Tools (כלים)**
- Skills = יכולות מיוחדות (פוסטים ללינקדאין, ניוזלטרים...)
- Workflows = תהליכים מרובי שלבים שחוזרים על עצמם
- Prompts = הוראות לשימוש חוזר למשימות נפוצות

**O = Output (תוצרים)**
- כל מה שהסוכנים יוצרים
- מאורגן. מגורסא. קל למצוא.

**M = Memory (זיכרון)**
- Learning Log = מה עבד, מה לא, למה
- פידבק = התיקונים שלכם הופכים לשיפורים עתידיים
- החלטות מתועדות = קונטקסט לבחירות עתידיות
- זה מה שגורם למערכת להשתפר עם הזמן

## מה בנינו היום

**צעד 1:** מילאנו את השאלון להגדרת המותג כדי ליצור את קבצי ה-C-core שלכם

**צעד 2:** הוספנו דוגמאות כתיבה כדי להאכיל את ה-B-brain שלכם

**צעד 3:** יצרנו את הפוסט הראשון ללינקדאין (כדוגמה לאיך להשתמש בכל המערכת)

התוצאה: צוות AI שמכיר את המותג שלכם וכותב כמוכם.

**היום למדנו:**
- מערכת ABC-TOM
- איך להאכיל את ה-Brain

- איך לגרום לסוכנים לקרוא הכל

בתיקייה `T-tools/prompts/` תמצאו:

## טיפים מהשטח

- תהיו ספציפיים. "תכתוב לי פוסט" פחות טוב מ"תכתוב לי פוסט על X לקהל של Y."
- תנו דוגמאות. רוצים סגנון מסוים? תראו להם.
- תסמכו על הגייטקיפר. הוא שומר על הסטנדרט.
- אחד הסוכנים פישל? תנו לו פידבק טוב ותבקשו שיעדכן את עצמו בהתאם.
- אל תפחדו לשנות. המסמכים שלכם. תעדכנו אותם.
- **פרויקט חדש? שכפלו את התיקייה.** רוצים לכתוב ספר? לעבוד עם לקוח ספציפי? פרויקט צד? פשוט שכפלו את תיקיית ABC-TOM (תנו לה שם אחר כמובן שיתאים לפרויקט) ותתחילו למלא מחדש. רק תוודאו שהתיקייה נשארת בתוך ה-Vault של Obsidian כדי שקלוד ימשיך לדבר איתה.
